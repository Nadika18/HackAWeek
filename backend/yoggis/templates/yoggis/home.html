What is exactly Yoggis?

Yoggis is a personalized, on-demand yoga instruction app from a virtual AI assistant that provides real-time feedback.

How does it work?

The machine learning model calculates the accuracy of the yoga postures based on some calculations at the back-end.

Can anyone use it?

Yes! Anyone sitting at their homes can access the website with just a click.
